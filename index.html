JavaScript Array Methods Tasks

Complete the tasks below by writing JavaScript code. Use the sample arrays provided for each task.

Task 1: Iteration and Transformation

      Sample Array: [1, 2, 3, 4]
      1. map(): Result should look like [2, 4, 6, 8]
      2. forEach(): Each name should be logged individually (e.g., "Alice", "Bob", "Charlie")
      3. filter(): Result should look like [6, 7, 8]
      4. reduce(): Result should look like 10
      5. find(): Result should look like { id: 1, name: "John" }
      6. findIndex(): Result should look like 2
      7. some(): Result should look like true/false
      8. every(): Result should look like true/false

Task 2: Adding and Removing Elements

      Sample Array: [1, 2, 3]
      1. push(): Array should look like [1, 2, 3, 4]
      2. pop(): Removed value should be 4, array should look like [1, 2, 3]
      3. unshift(): Array should look like [0, 1, 2, 3]
      4. shift(): Removed value should be 0, array should look like [1, 2, 3]
      5. splice(): Array should look like [1, 4, 5] after removing and [1, 2, 3, 4, 5] after inserting
      6. slice(): Result should look like [1, 2, 3]

Task 3: Searching and Sorting

      Sample Array: [4, 3, 2, 1]
      1. indexOf(): Result should look like 1
      2. lastIndexOf(): Result should look like 3
      3. includes(): Result should look like true/false
      4. sort(): Result should look like [1, 2, 3, 4]
      5. reverse(): Result should look like [4, 3, 2, 1]

Task 4: Joining and Splitting

      Sample Array: ["Hello", "world"]
      1. join(): Result should look like "Hello world"
      2. concat(): Result should look like [1, 2, 3, 4, 5, 6]

Task 5: Flattening and Mapping

      Sample Array: [1, [2, [3, 4]], 5]
      1. flat(): Result should look like [1, 2, [3, 4], 5] or [1, 2, 3, 4, 5]
      Sample Array: ["Hello"]
      2. flatMap(): Result should look like ['H', 'e', 'l', 'l', 'o']

Task 6: Utility Methods

      Sample Array: [1, 2, 3]
      1. Array.isArray(): Result should look like true/false
      Sample String: "Hello"
      2. Array.from(): Result should look like ['H', 'e', 'l', 'l', 'o']
      3. Array.of(): Result should look like [1, 2, 3]
